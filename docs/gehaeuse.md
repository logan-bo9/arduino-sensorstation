# Gehäuse

Für die Station wurde ein Gehäuse in Tinkercad entworfen und per 3D-Druck
gefertigt. Es bietet Aussparungen für das Display, die LEDs, die Taster
und die Sensoren sowie eine feste Befestigung für den internen Aufbau.

## Anforderungen

- Ausschnitt für das LCD-Display
- Öffnungen für die drei LEDs
- Zugänge für die vier Taster
- Öffnungen für LDR und Sound-Sensor
- Stabile Befestigung der Platine im Inneren
