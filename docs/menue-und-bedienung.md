# Menü und Bedienung

Die Station wird über vier Taster bedient: UP, DOWN, OK und BACK.

## Startbildschirm

Zeigt den Namen der Station und den aktuellen Status.

```text
ENV STATION
STATUS: OK
```

## Sensorübersicht

Mit UP und DOWN wechselt man zwischen den vier Sensoren:

- TEMP
- HUM
- LIGHT
- SOUND

## Detailansicht

Zeigt den aktuellen Messwert und den zugehörigen Grenzwert.

```text
TEMP: 23C
LIM: 30
```

## Grenzwert bearbeiten

1. Sensor mit UP/DOWN auswählen
2. Mit OK die Bearbeitung öffnen
3. Mit UP/DOWN den Grenzwert ändern
4. Mit OK speichern oder mit BACK abbrechen

```text
EDIT LIMIT
T: 30
```

## Tastenbelegung

| Taste | Funktion |
|---|---|
| UP | Nach oben navigieren / Wert erhöhen |
| DOWN | Nach unten navigieren / Wert verringern |
| OK | Auswählen / Bearbeiten öffnen / Speichern |
| BACK | Zurück / Abbrechen |
