# Arduino-Sensorikstation

Arduino-Lernprojekt zur praktischen Integration von Temperatur-, Feuchte-,
Helligkeits- und Geräuschsensorik mit Anzeige, Benutzerführung und
konfigurierbarer Warnlogik.
